# webzh-i-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容彼此独立，不针对任何特定域名或具体网站。仓库的主要目的是提供一个集中存放和展示 HTML 资料的空间。

## 项目简介

webzh-i-archive 是一个轻量级的页面归档项目。所有内容均以静态 HTML 文件形式存在，无需后端服务即可访问。适合存放各类信息型页面、文档快照或临时展示内容。

## 目录说明

```
/
├── index.html          # 仓库入口页面，列出所有归档页面链接
├── pages/              # 归档页面存放目录
│   └── ...             # 各独立 HTML 文件
└── README.md           # 本文件
```

- `index.html`：主页，提供导航功能，方便浏览所有归档页面。
- `pages/`：所有归档的 HTML 页面均放置在此目录下，文件名通常反映内容主题。

## 页面归档说明

- 每个页面均为完整独立的 HTML 文件，包含自身样式与结构。
- 归档内容可能来自公开资料、个人整理或临时记录，仅供存档参考。
- 不保证所有页面内容持续更新或保持最新。
- 页面中不包含任何外部跟踪、广告或交互式脚本。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面会通过提交记录体现。
- 如需引用或复制页面内容，请自行判断其适用性。
- 仓库不提供内容审核或修改请求的保证，但欢迎通过 Issues 讨论合理建议。
- 所有归档页面仅作为资料留存，不代表维护者立场或观点。

## 使用方式

可直接通过 GitHub Pages 或本地浏览器打开 `index.html` 浏览所有页面。若需单独访问某个页面，直接打开 `pages/` 下对应文件即可。
